# Workshop in Digital Culture Class Practice Branch

Student Guide — Submitting Your Files 
This guide explains how to upload your files  into your personal folder inside the class-practice branch.

## Step 1. Open the class repository
Go to: https://github.com/MonirehRazavi/Workship-in-DigitalCulture

## Step 2. Switch to the correct branch
Use the branch dropdown → select class-practice.

## Step 3. Navigate to your folder
Click through: class-practice/FirstName-LastName/

## Step 4. Upload your files
Click Add file → Upload files
Then drag/drop your fies.

## Step 5. Commit your changes
At the bottom:

Write a short commit message (e.g., “Adds raw and cleaned CSV files”, for example)
Click Commit changes

## Step 6. Verify upload
Reload the page and check that your files appear in your folder.
